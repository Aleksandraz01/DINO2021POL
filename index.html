git init - To polecenie stworzy nowy podkatalog o nazwie .git, zawierający wszystkie niezbędne pliki — szkielet repozytorium Gita.

git config --global user.name "" - polecenie ustawia nazwę użytkownika (--global oznacza, ze konfiguracja jest dokonywana jednorazowo) Anna Śmigiel

git git config --global user.email "" - polecenie ustawia adres email użytkownika (--global oznacza, że konfiguracje wystarczy dokonać jednorazowo) Maksymilian Kulikowski

git checkout -b "branch" - polecenie tworzy nowego brancha o podanej nazwie i przenosi nas do niego. Filip Nowicki

git pull "remote" "branch" - polecenie zaciaga zmiany z repozytorium, aby lokalna wersja repozytorium byla aktualna, gdzie remote to nazwa remote'a (np, origin) a branch to nzawa brancha (np. master)
