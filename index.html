Hello World

git clone /path/to/repository - klonuje repozytorium z podanej ścieżki do obecnego folderu na dysku

git config --global user.email "" - Komenda służy do konfiguracji adresu email. Opcja --global zmienia adres email dla wszystkich projektów git.

Git push jest stosowany w celu publikowania nowych lokalnych commitów do zdalnego repozytorium. Opcja "remote branch" pozwala na wybranie brancha, na który chcemy wysłać zmiany.
