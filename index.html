Hello World

git clone /path/to/repository - klonuje repozytorium z podanej ścieżki do obecnego folderu na dysku

git config --global user.email "" - Komenda służy do konfiguracji adresu email. Opcja --global zmienia adres email dla wszystkich projektów git.

git config --global color.ui true -> Komenda ustawiona na wartość true koloruje informacje zwrotne przeznaczone dla użytkownika w terminalu.

git commit "filename" -> zapisuje migawkę plików naszego projektu. Commit zawiera wszystkie pliki projektu. W przypadku plików, które się nie zmieniły, git przechowuje tylko łącze do poprzedniego identycznego pliku, który jest już zapisany.
